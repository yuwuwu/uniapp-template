# uniapp-template

## 安装依赖
npm install/yarn install

## 运行到h5
npm run dev:h5-dev(测试环境)
npm run dev:h5-prod(正式环境)

## 运行到微信小程序

npm run dev:wx-dev(测试环境)
npm run dev:wx-prod(正式环境)

## 打包到h5
npm run build:h5-prod

## 打包到微信小程序
npm run build:wx-prod
